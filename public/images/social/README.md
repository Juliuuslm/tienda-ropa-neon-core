# Social / Comunidad Section Images

## Recommended Images

### Social Grid (4 images)

#### Images 1-4
- **Filename**: `social-1.jpg`, `social-2.jpg`, `social-3.jpg`, `social-4.jpg`
- **Dimensions**: 400x400px (1:1 square)
- **Purpose**: Instagram-style showcase of NEON CORE community
- **Grid Layout**: 4-column grid (responsive to 2 columns on tablet, 1 on mobile)

## Effects Applied
- Grayscale by default
- Zoom on hover (scale 110%)
- Rotate slightly on hover (rotate-2)
- Dark overlay (cyan tinted) appears on hover
- Instagram icon appears on hover (animated bounce)

## Content Ideas
- Behind-the-scenes photos
- Customer styling (user-generated content)
- Product details
- Event/collaboration shots
- Street fashion featuring NEON CORE gear

## Usage in Component
Images mapped in social grid array

## Notes
- Aspect ratio must be 1:1 (square)
- High contrast images work best with the grayscale effect
- Consider slightly under-exposed images for better hover reveal effect
- All 4 images should have consistent lighting/style for cohesive look
